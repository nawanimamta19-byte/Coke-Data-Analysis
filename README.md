# 🥤 Coca-Cola Sales Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing Coca-Cola sales performance across multiple regions, retailers, and beverage brands using Microsoft Excel. The dashboard provides valuable business insights into sales trends, profitability, and operational performance through interactive visualizations and KPI tracking.

The goal of this project is to transform raw sales data into actionable insights that can support strategic business decisions and improve overall sales performance.

---

## 🎯 Business Objectives

✅ Analyze overall sales performance

✅ Track operating profit and profitability trends

✅ Compare retailer-wise and brand-wise sales contribution

✅ Identify top-performing regions and products

✅ Create an interactive dashboard for business reporting
